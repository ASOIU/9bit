Jetbytes: сервис передачи файлов

Jetbytes — это экспериментальный веб-сервис для передачи файлов без промежуточного хранения на сервере.

Многие используют файл-хостинги для отправки файла единственному человеку. После успешной передачи потребность хранения файла на хостинге становится ненужной, а иногда и опасной.

Отличительный принцип сервиса: файл по пути нигде не хранится (кроме серверов спецслужб :) ). Это позволяет (теоретически) передавать файл любого размера. Не нужно ждать, пока файл загрузится на сервер - ссылка на закачку доступна сразу после выбора файла. Её нужно передать получателю и, не закрывая вкладку браузера, дождаться окончания передачи. POST-запрос из браузера отправителя работает параллельно с GET-запросом получателя.

Плюсы:

  • минимум действий: отправка файла проще, чем по почте;
  • не нужно ждать завершения Upload'а;
  • суммарное время передачи примерно вдвое меньше, по сравнению с передачей через файл-хостинг;
  • есть возможность наблюдать за процессом передачи на протяжении всего процесса (и на 98% передачи файла со словами "Ха-Ха" закрыть окно браузера :( ).

Минусы:

  • несовместимость с некоторыми менеджерами закачек, которые намеренно рвут соединение по ходу закачки (ссылка одноразовая).
