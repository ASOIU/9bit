Региональная – 2010

28-29 апреля 2010 г. в нашем университете состоится Региональная студенческая олимпиада по программированию. Олимпиада проводится в виде личного первенства с автоматической проверкой задач с помощью системы Contester. Разрешённые языки программирования: Pascal, C++, Java, C#, VB.NET. Заявки принимаются до 27 апреля, 12:00. Команда ОмГТУ будет сформирована автоматически по шести лучшим результатам университетской олимпиады. Подробности тут.
