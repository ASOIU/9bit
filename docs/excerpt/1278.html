Поиск картинок по цвету

Гуляя по просторам Интернета, случайно наткнулся на интересный сайт http://labs.ideeinc.com/multicolr/# . Это сервис, позволяющий искать картинки в интернете по заданным пользователем цветам. Проще говоря - выбираем несколько цветов (от 1-го до 10-ти), и сервис выдает нам картинки, в которых эти цвета преобладают. Причем у каждого цвета существует несколько вариантов, отличающихся по насыщенности и оттенку, поэтому вариативность результата максимальна.
