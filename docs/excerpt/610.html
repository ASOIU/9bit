Хорошие бесплатные программы для Windows

Существует множество shareware-программ из разряда must have. К сожалению, мало кто платит за эти программы, все просто скачивают кряки. Уровень пиратства в нашей стране довольно высок.

В свете последних событий особенно жестко стоит вопрос о лицензионности установленных программ. Многие видят будущее за свободной лицензией и переходом на Linux. Но далеко не все готовы так глобально менять свои привычки. Windows еще надолго останется лидирующей операционной системой. Но в наших силах снизить уровень пиратства, ведь помимо самой системы пользователи устанавливают большое количество платных программ, часто обременненных излишней функциональностью.

Чтобы использовать бесплатные программы, не нужен переход на другую систему – под Windows есть огромное количество бесплатного и открытого программного обеспечения. В рамках проекта freealt.net отбираются самые лучшие из бесплатных программ, постепенно заменяя взломанные аналоги.
