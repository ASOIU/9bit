Google Docs, OpenOffice.org

OpenOffice + Google Docs

Для пользователей OpenOffice и Google Docs!

Интересное расширение к OpenOffice - OpenOffice.org2GoogleDocs.

Это расширение для OpenOffice позволяющее импротировать/экспортировать документы из Google Docs, Zoho, а также экспортировать на серверы WebDAV. Поддерживаются следующие типы документов:

  • OpenDocument Text (.odt)
  • StarOffice (.sxw)
  • Microsoft Word (.doc, формат офиса 2007 - .docx не поддерживается)
  • Rich Text (.rtf)
  • OpenDocument Spreadsheet (.ods)
  • Microsoft Excel (.xls)
  • Comma Separated Value (.csv)
  • Microsoft PowerPoint (.ppt, .pps)

Скачать расширение можно тут: http://extensions.services.openoffice.org/download/1888
Подробности можно прочитать тут: http://truediamon.ru/content/openoffice-google-docs
