Результаты ЕГЭ по информатике и ИКТ – 2009

Пара слов о том, как мы проверяли результаты ЕГЭ по информатике и ИКТ. Процедура следующая. Омский «Региональный информационно-аналитический центр» (РИАЦ, www.omedu.ru) сканирует работы школьников с помощью системы ABBYY TestReader Network, которая сразу же проверяет ответы секций A и B. В день экзамена вечером экспертная комиссия получает правильные ответы секции C с подробными инструкциями по оценке (для всех вариантов), и следующие два дня уходят на «ручную» проверку развёрнутых ответов секции C.
