Тренировки к олимпиаде по программированию

Кафедра АСОИУ приглашает всех желающих принять участие в тренировках по программированию (для подготовки к университетской олимпиаде, которая состоится ориентировочно 28 апреля): каждый понедельник с 13:15 до 14:45 или с 14:50 до 16:20, ауд. 8-205. Для автоматической проверки задач используется система Contester, которая легко разворачивается локально и не требует никакой дополнительной настройки. Контестер содержит в себе готовые сборники задач. На тренировках используется сборник «Фёдор Меньшиков. Тренировки».
