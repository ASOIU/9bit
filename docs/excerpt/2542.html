Простой способ оформить "список источников" по ГОСТу

  1. Открываем нужную статью.
  2. В левой колонке "Инструменты" щелкаем на "Цитировать страницу".
  3. Копируем Стиль по ГОСТ 7.0.5—2008 (библиографическая ссылка).
  4. Читаем "Примечание" и правим даты.
