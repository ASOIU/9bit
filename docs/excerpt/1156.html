Кружок по программированию «Дебаггер»

Программирование на Pascal, C++ и C#, решение олимпиадных задач (для желающих), изучение основных алгоритмов, разбор задач ЕГЭ по Информатике и ИКТ. Продвинутые школьники могут начать изучать Java, cистемное программирование под Windows и Linux, web-программирование (от JavaScript до PHP и ASP.NET).

Каждую субботу, 15:00 (день недели и время меняется по договоренности участников). Предварительная запись: по электронной почте или по телефону 65-27-94. См. также раздел «Кружки для школьников» нашего форума.
