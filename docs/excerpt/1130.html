Тренировки по программированию

Продолжаются тренировки по программированию. Решаем олимпиадные задачи, разбираем алгоритмы. Для передовиков – возможность подготовиться к олимпиадам. Для новичков – возможность подтянуть программирование. Кроме наших приглашаем студентов других кафедр, желающих войти в команду университета для участия в региональных олимпиадах по программированию.
